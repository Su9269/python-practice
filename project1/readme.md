# NBA 2024-25 球員分析：分群與季後賽預測

## 專案動機
本人對數據分析有濃厚的興趣，會想要了解數據背後所代表的意義以及透過數據去了解我們好奇的事情，因此我選擇我平常有在關注的NBA(美國職籃)進行分析，探討像是球員在場上扮演的角色或是對球隊貢獻的多寡除了基礎數據外，也拿了進階數據一併做預測，進階數據更能代表球員在場上的價值以及效用，這個專案希望透過統計降維與分群方法，找出球員在數據上呈現的「球風類型」，並進一步嘗試用球員個人數據預測球隊能否打進季後賽。

## 資料說明
資料來源為NBA 2024-25球季四份原始數據：基礎數據、進階數據、薪資、季後賽結果。四份資料以球員姓名合併，經清理後共569筆、44個欄位。（資料來源、四份CSV、合併後569筆x44欄）

## 資料清理:
## Salary(薪資)：
原始資料含有 `$`與千分位逗號等符號，需先清除後轉換為數值型態。清理後仍有82筆缺失，經檢查發現這批球員分散於各隊、與轉隊無關，判斷為個別合約類型特殊（如雙向合約）所致。考量薪資本身與位置、上場時間高度相關，最終以「同位置（Pos）薪資中位數」補值，而非以得分等表現數據推算，避免人為製造得分與薪資之間不存在的相關性，影響後續PCA分析。
### 命中率類欄位（FG%, 3P%, 2P%, eFG%, FT%, TS%）
缺失值經查證，皆因對應出手數（FGA/3PA等）為0，屬於「分母為0導致無法計算」，而非資料缺漏，因此直接補0。
### 進階數據異常（PER, WS, BPM 等10個欄位）
發現2位球員（Nikola Jović、Skal Labissière）此類欄位整批缺失。經查證：
- **Nikola Jović**：因姓名含特殊字元（ć），與進階數據來源表合併時字串沒有完全匹配，導致資料遺失。手動查證原始數據源後補上正確數值。
- **Skal Labissière**：出賽場次僅4場（12分鐘），樣本量過小，進階指標不具統計意義，所以移除該筆資料。
### PLAYOFF（季後賽結果）
81筆缺失，查證後100%對應「賽季中轉隊」（Team欄位為2TM/3TM）球員。由於轉隊球員的季後賽歸屬定義本身模糊，這批資料不參與模型訓練，保留作為訓練完成後的獨立預測與人工驗證對象。

## EDA（探索性資料分析）:
### 相關係數分析
多個變數之間存在高度相關（相關係數 > 0.98），例如：
- `PTS` 與 `FG`（0.994）、`FGA`（0.992）
- `3P` 與 `3PA`（0.992）、`2P` 與 `2PA`（0.991）
- `DRB` 與 `TRB`（0.986）
這些相關性多屬「總量與分量」的結構性重複（如TRB = ORB + DRB）。考量PCA本身即設計用於處理具相關性的變數，最終保留全部連續型變數進行降維，讓PCA自行處理共線性問題。
### 分佈檢視
多數量化指標（PTS, AST, TRB, MP, WS）呈現明顯右偏分佈，反映聯盟球員在上場時間與得分機會上的結構性落差（少數主力球星 vs 多數輪替球員）。僅PER（效率值）呈現近似常態分佈，因其計算方式已對出場時間做標準化調整，較不受「量」的落差影響。
### 位置間數據差異
以Pos分組後，PG（控球後衛）的平均助攻數（205）遠高於其他位置（90-110區間），阻攻數（14）則最低；C（中鋒）的平均籃板（320）與阻攻（44）皆為最高，助攻（89）則墊底。此現象顯示位置本身即為影響球員數據結構的重要因子，也是後續K-means分群結果需要對照驗證的基準。

## PCA 降維:
以37個原始連續型變數（排除Player, Team, Pos, Salary, PLAYOFF, Rk）進行標準化後，執行PCA降維。
### 主成分數量選擇
綜合兩種方法判斷：
- **凱瑟準則**（特徵值 > 1）：建議保留5個主成分
- **累積解釋變異量**：5個主成分累積解釋 84.4% 的原始變異量，達到實務上可接受的門檻
兩種方法結論一致，最終保留5個主成分。
### 主成分意義解讀
透過檢視各主成分的loadings（原始變數對主成分的貢獻權重），解讀出5個球風主成分：
| 主成分 | 解釋變異量 | 代表意義 |

| PC1 | 54.4% | 整體出場量／產出量（主力球員 vs 邊緣輪替球員） |
| PC2 | 13.4% | 效率型（高命中率、低出手量）vs 數量型（高出手量、高球權） |
| PC3 | 7.2% | 外線射手型（三分出手多、命中率高）vs 內線苦工型（籃板、阻攻多） |
| PC4 | 5.3% | 球星型（高使用率、高效率、高替代價值）vs 場次型角色球員 |
| PC5 | 4.0% | 防守見長（防守正負值高）vs 進攻主導（高球權使用率） |
PC4、PC5解釋力相對較低，訊號較弱，與其他主成分略有重疊，屬正常現象。


## K-means 分群:
### K值選擇
綜合兩種方法評估：
- **手肘法**：群內誤差（inertia）在K=4之後下降幅度明顯趨緩（K=3→4下降1554，K=4→5降幅腰斬至628）
- **輪廓係數**：K=2分數最高（0.397），但呈單調遞減趨勢
兩種方法出現分歧：輪廓係數指向K=2（分群界線最清楚），手肘法指向K=4（邊際效益轉折點）。考量本專案目的為探索球員的球風類型分類，K=2可能僅反映最粗略的「主力vs板凳」二分，分析價值有限；而K=4在輪廓係數上（0.316）與K=5（0.317）幾乎持平,未明顯犧牲分群品質，最終選擇K=4。
### 分群結果解讀
族群主要由PC1&PC4主宰，透過觀察這兩個主成分比較有故事性
| 群組 | 人數 | PC1 | PC4 | 代表意義 |

| 群0 | 195 | 中等偏高 | 最高（正） | 穩定輪替的資深角色球員 |
| 群1 | 62 | 最低（負） | 偏負 | 板凳末端／邊緣球員 |
| 群2 | 75 | 最高（正） | 最低（負） | 聯盟頂級球星 |
| 群3 | 236 | 偏負 | 中性 | 效率型外線射手 |
### 侷限反思
四群結果主要受PC1（整體產出量,解釋54.4%變異）主導,PC5（防守vs進攻,僅解釋4%變異）訊號相對微弱，導致防守專精但整體數據量不特別突出的球員（如防守悍將），未能獨立成群，而是被歸入「頂級球星」類別（因其整體數據量本身也高，且透過手動查找也證實）。這是本次分群方法基於數據結構的自然結果，也是後續可延伸探討的方向。

## Random Forest 季後賽預測:
### 資料切分
488筆具PLAYOFF標籤的資料用於訓練與測試（80/20切分），81筆賽季中轉隊球員因季後賽歸屬定義模糊，PLAYOFF為缺失值，保留作為訓練完成後的獨立預測對象，不參與訓練與模型評估。
### 特徵選擇
使用37個原始數值欄位作為特徵，排除Salary（市場評價,非球場表現數據）與PCA衍生欄位（PC1-5, Cluster，避免同一資訊被重複輸入模型）。
### 模型表現與調整
初始模型（未調整類別權重）出現偏向多數類別的傾向，對「未進季後賽」球員的召回率僅0.60。加入 `class_weight='balanced'` 平衡類別權重後：
| 指標 | 調整前 | 調整後 |

| Accuracy | 72.4% | 74.5% |
| 0類別 Recall | 0.60 | 0.70 |
調整後模型在整體準確率與少數類別辨識能力上皆有提升。
### 特徵重要性
模型判斷季後賽晉級與否時，最倚重的前5項特徵依序為 DWS、DBPM、WS/48、WS、OBPM，皆屬防守效率與綜合貢獻指標，而非單純得分、籃板等基礎數據。此結果反映兩層意義：
1. **統計層面**：進階指標本身是基礎數據的加權整合，蘊含資訊量高於單一原始欄位
2. **領域知識層面**：現代NBA進攻效率普遍提升，得分火力已非決定勝負的主要區分因子，防守穩定性反而是球隊能否累積勝場的關鍵變因

### 轉隊球員預測驗證
以訓練完成的模型對81筆轉隊球員進行預測，並人工查證部分知名案例的實際結果：
| 球員 | 交易去向 | 球隊實際結果 | 模型預測 | 結果 |

| Luka Doncic | 湖人 | 進季後賽 | 進季後賽 | 正確 O |
| Anthony Davis | 獨行俠 | 未進季後賽 | 進季後賽 | 錯誤 X |
| Jimmy Butler | 勇士 | 進季後賽 | 進季後賽 | 正確 O |
| Andrew Wiggins | 熱火 | 進季後賽 | 進季後賽 | 正確 O |
| Dennis Schröder | 活塞 | 進季後賽 | 未進季後賽 | 錯誤 X |

值得注意的是，Anthony Davis與Luka Doncic為同一筆交易案的兩位主角，模型對兩人皆預測「進季後賽」，但實際結果一對一錯。這顯示模型的核心侷限：預測僅基於「球員個人數據」，無法納入球隊整體陣容、傷病、化學反應等團隊層級因素，而季後賽晉級終究是「球隊」而非「個人」的結果。

### 延伸思考
本次以37個原始球員數據欄位作為訓練特徵。若改以PCA降維後的5個主成分作為特徵，將能從「球風類型」角度（而非單一數據指標）觀察哪種類型的球員對球隊晉級季後賽貢獻較大，並與K-means分群結果相互呼應，是後續可延伸的分析方向。
